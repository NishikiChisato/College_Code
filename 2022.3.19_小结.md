**求出区间内的奇数**

```cpp
int countOdds(int low, int high)
    {
        if(low&1)
            return (high - low) / 2 + 1;
        else
            return (high - low + 1 ) / 2;
    }
```

`右移相当于除以2的次方，左移相当于乘2的次方`
```cpp
int per(int x)//求出(x+1)/2向下取整的值
{
    return (x + 1) >> 1;
}
int countOdds(int low, int high)//区间0到x内的奇数个数为per(x)
    {
        return (per(high) - per(low - 1));
    }
```

**一行代码实现斐波那契数列**

```cpp
int fib(int x)
{
    return x <= 1 ? 1 : fib(x - 1) + fib(x - 2);
}
```
